Build a B2B SaaS portal with the following:

1. Authentication (email/password + SSO via SAML)
2. Role-based access control (3 roles: owner, admin, viewer)
3. Dashboard showing key metrics
4. Stripe subscription billing (3 plans: Starter, Pro, Enterprise)
5. Audit log of all admin actions
6. User management (invite, role change, deactivate)
7. Settings page (org settings + user settings)
8. Email notifications (welcome, invite, billing)
9. Production-ready (deployable, secure, monitored)

Tech stack: any modern web framework, Postgres, deployable to a container platform. Include CI/CD config. Output complete codebase, ready to deploy.
